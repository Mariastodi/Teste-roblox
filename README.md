# Estudo de Movimentação e TweenService no Roblox

Este repositório contém os arquivos de um projeto prático realizado para fins de **estudo de desenvolvimento de jogos**. O objetivo principal foi explorar a manipulação de objetos no espaço 3D e a criação de sistemas de transporte automatizados utilizando a biblioteca `TweenService` do Roblox.

## Sobre o Projeto
O projeto consiste em uma plataforma flutuante que conecta duas ilhas em um ambiente 3D. A plataforma realiza um movimento suave e contínuo, ajustado para percorrer uma distância específica nos eixos X e Y, garantindo que o jogador consiga transitar entre as áreas do mapa.

### Tecnologias Utilizadas:
* **Luau**: Linguagem de script do Roblox.
* **TweenService**: Utilizado para criar animações fluidas baseadas em propriedades (CFrame).
* **Roblox Studio**: Engine de desenvolvimento.
